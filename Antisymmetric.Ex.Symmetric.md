A symmetric binary operator gives rise to an antisymmetric relation 🤯

```
●	Antisymmetric ⊑

1.	x ⊑ y  ≡  x ⊓ y = x

2.	Symmetric ⊓

≡		Definition of Antisymmetry

	∀(x,y ∈ A • x ⊑ y  ⋀  y ⊑ x  ⇒  x = y)

≡		Establish ∀

	●	x ⊑ y  ⋀  y ⊑ x

	≡		Assumption 1.

		x ⊓ y = x  ⋀  y ⊓ x = y

	≡		Assumption 2.

		x ⊓ y = x  ⋀  x ⊓ y = y

	⇒		= Transitive

		x = y

	▢

⋯	∀(x,y ∈ A • true)

≡		Trivial

	true

▢
```
