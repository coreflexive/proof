# WP Monotonic

```
●	(X ⇒ Y)  ⇒  (wp.S.X  ⇒  wp.S.Y)

⊩	< Prove ⇒ >

	●	wp.S.X  ⇒  wp.S.Y

	1.	X ⇒ Y

	a.	X ⋀ Y ≡ X	< Implication on 1. >

	≡		< Implication >

		wp.S.X ⋀ wp.S.Y  ≡  wp.S.X

	≡		<WP Conjunctivity>

		wp.S.(X ⋀ Y)  ≡  wp.S.X

	≡		< Observation a. >

		wp.S.X  ≡  wp.S.X

	≡		< Reflexivity of ≡ >
	
		true

	▢

▢
```
